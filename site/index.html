Firefly Synth

An open-source semi-modular VST3 and CLAP synthesizer and FX plugin for Windows, Linux and Mac.

Download

See github releases.

Support

Mail me or (preferably) open a github issue.

Requirements

Windows: 7+, X64, AVX1 support.
MacOS: 10.15+, X64 (Intel) or 64-bit ARM (Silicon).
Linux: Ubuntu 18.04+, X64, AVX1 support (known to work on other distros, actual requirement is glibc 2.25+).
