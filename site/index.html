Firefly Synth

Open-source semi-modular VST3 and CLAP synthesizer and FX plugin for Windows, Linux and Mac.

Install

Download the zip file and copy it's contents (all of it, not just the dll/so/dylib) to you CLAP/VST3 folder.

System requirements

Windows: 7+, X64, AVX1 support.
MacOS: 10.15+, X64 (Intel) or 64-bit ARM (Silicon).
Linux: Ubuntu 18.04+, X64, AVX1 support (known to work on other distros, actual requirement is glibc 2.25+).
